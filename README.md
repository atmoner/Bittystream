# Bittystream  

Install:  
`wget https://github.com/atmoner/Bittystream/archive/master.zip && unzip master.zip && cd Bittystream-master/ && npm install`

Run:  
`node index.js`

Now you can stream your torrent on port 3000   
`http://domain.dot:3000`
